HTML

HTML, ou HyperText Markup Language, é a linguagem de marcação padrão para criar a estrutura de páginas da web e o conteúdo que nelas aparece, como parágrafos, títulos, imagens e links

CSS

CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem utilizada para definir a apresentação visual de documentos escritos em HTML ou XML, como sites.
